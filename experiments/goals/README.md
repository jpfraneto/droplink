# Experiment Goals

Create one folder per goal using:

```text
YYYY-MM-DD-short-name/
```

Recommended files inside a goal folder:

- `brief.md`: hypothesis, target behavior, and success criteria.
- `plan.md`: steps, commands, and rollback notes.
- `observations.md`: what happened, with links to any run artifacts.
- `decision.md`: what to keep, discard, or promote into production code.

Keep bulky generated output in `../runs/` or `../results/`, then link to it from the goal notes.
