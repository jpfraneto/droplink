# Product Print Prompt Template

Use this template for one selected Printful product.

```text
Create print-ready artwork for {{brandName}}, one relic in a cohesive three-object DropLink triptych.

Selected product:
- Product name: {{productName}}
- Variant: {{variantName}}
- Catalog product id: {{catalogProductId}}
- Catalog variant id: {{catalogVariantId}}
- SKU: {{sku}}
- Placement: {{placement}}
- Technique: {{technique}}

Triptych:
- Position: {{relicIndex}} of 3
- Relic name: {{relicName}}
- Role: {{roleInTriptych}}
- Object function: {{objectFunction}}
- Description: {{description}}
- Why this exists: {{whyThisExists}}

Brand:
- Thesis: {{brandThesis}}
- Archetype: {{brandArchetype}}
- Worldview: {{worldview}}
- Emotional posture: {{emotionalPosture}}

Shared visual system:
- Shapes: {{shapes}}
- Motifs: {{motifs}}
- Materials: {{materials}}
- Composition rules: {{compositionRules}}
- Palette: {{palette}}
- Signature gesture: {{signatureGesture}}

Art direction:
{{artDirection}}

Make this artifact visually sibling to the other two relics: shared palette, shared symbolic grammar, distinct role. Use precise, print-friendly geometry and intentional negative space. If using words, use only short original phrases from the phrase bank.

Avoid:
{{avoidList}}

Hard constraints:
- No product mockup body in the print file.
- No background scene.
- No exact unauthorized logos.
- No celebrity likenesses.
- No dense copy.
- No website screenshots.
```
